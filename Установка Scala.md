Детали можно найти на странице проекта https://scala-lang.org/download/

Устанавливать Scala рекомендуется с помощью утилиты `cs`, которую на MacOS можно установить с помощью менеджера `brew`
```bash
$ brew install coursier/formulas/coursier && cs setup
```
По умолчанию ставится Scala 3. Обычно это не вызывает никаких проблем, потому как в конкретном проекте можно указать требуемую версию Scala. 

В случае необходимости можно установить Scala 2 ветки и использовать ее по умолчанию
```bash
$ cs install scala:2.13.14 scalac:2.13.14
```
